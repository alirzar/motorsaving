# Archived pipeline for Rezaei et al. 2026 (PLOS Biology)

This repository contains the code (and pointers to data) needed to reproduce the analyses and figures in:

### Rezaei et al., 2026. “The retrieval of previously learned motor memories is facilitated by the reinstatement of default mode network manifold structures” (PLOS Biology).

## Setup
Clone the repository and create the environment:

```bash
git clone https://github.com/alirzar/motorsaving.git
cd motorsaving
conda env create -f environment.yml
conda activate motorsaving
pip install -e .
